2  Database schema

2.1 Main hscic tables

The hscic dataset has 194 different tables. Here we describe the main tables used for research projects. To see all the tables available in the hscic dataset, you can use:

dbListTables(con)

2.1.1 normalised_prescribing

Normalised view of the main prescribing dataset, updated monthly. This is the table you’ll typically want to query rather than the raw prescribing table.

Columns
sha character Strategic Health Authority code (e.g., QHM)
regional_team character Regional team identifier (e.g., Y63)
stp character Sustainability and Transformation Partnership code (e.g., QHM)
pct character Primary Care Trust code (e.g., 16C)
practice character Practice code (e.g., A81001)
bnf_code character British National Formulary (BNF) code (e.g., 0408010W0BBAKA1)
bnf_name character British National Formulary (BNF) drug name (e.g., Epilim Chrono 300 tablets)
items integer Number of items (one item = one appearance on a prescription)
net_cost numeric Cost of the presentation according to the Drug Tariff
actual_cost numeric Actual cost including adjustments for bulk purchases, out of pocket expenses, etc. (This is the ‘cost’ field typically used in queries)
quantity numeric Number of pills/grams/millilitres/dressings/ampoules prescribed
month POSIXct Prescribing month
Rows 5 Columns 12

2.1.2 regional_teams

Open and close dates for regional teams.

Columns
code character Regional team code (e.g., Y57)
name character Regional team name (e.g., SOUTH OF ENGLAND COMMISSIONING REGION)
open_date Date Open date
close_date Date Closing date, NA if still open
Rows 5 Columns 4

2.1.3 practices

Includes all the practices in England, updated monthly. Contains latest month’s data only (but includes closed practices). Practices with a setting of 4 are standard GP practices.

Columns
code character Practice code (e.g., E82084)
name character Practice name (e.g., HATFIELD ROAD SURGERY)
address1 character Address line 1 (e.g., 13-15 RUSSELL AVENUE)
address2 character Address line 2 (e.g., LONDON COLNEY)
address3 character Address line 3 (e.g., LONDON COLNEY)
address4 character Address line 4 (town/city) (e.g., HATFIELD)
address5 character Address line 5
postcode character Practice postcode (e.g., AL1 3HD)
location character Location of practice in WGS84 format (e.g., 0101000020E6100000F24637A67312CDBF833AED8EDDE14940)
ccg_id character Clinical Commissioning Group identifier (e.g., 06N)
pcn_id character Primary Care Network identifier (e.g., U73966)
setting integer Practice setting code (0 = Other; 1 = WIC Practice; 2 = OOH Practice; 3 = WIC + OOH Practice; 4 = GP Practice; 8 = Public Health Service; 9 = Community Health Service; 10 = Hospital Service; 11 = Optometry Service; 12 = Urgent & Emergency Care; 13 = Hospice; 14 = Care Home / Nursing Home; 15 = Border Force; 16 = Young Offender Institution; 17 = Secure Training Centre; 18 = Secure Children’s Home; 19 = Immigration Removal Centre; 20 = Court; 21 = Police Custody; 22 = Sexual Assault Referral Centre (SARC); 24 = Other)
close_date character Date practice closed
join_provider_date character Date practice joined CCG
leave_provider_date character Date practice left CCG (or closed)
open_date character Date practice opened
status_code character Practice status code (A=Active, B=Retired, C=Closed, D=Dormant, P=Proposed, U=Unknown)
Rows 5 Columns 17

2.1.4 practice_statistics

Total list size and list sizes stratified by gender and age group for each practice, updated monthly. Contains latest month’s data only. Also contains calculated population weightings related to prescribing (STAR-PU and ASTRO-PU).

Columns
month POSIXct The month for which practice statistics are recorded
male_0_4 integer Count of male patients aged 0-4 years
female_0_4 integer Count of female patients aged 0-4 years
male_5_14 integer Count of male patients aged 5-14 years
male_15_24 integer Count of male patients aged 15-24 years
male_25_34 integer Count of male patients aged 25-34 years
male_35_44 integer Count of male patients aged 35-44 years
male_45_54 integer Count of male patients aged 45-54 years
male_55_64 integer Count of male patients aged 55-64 years
male_65_74 integer Count of male patients aged 65-74 years
male_75_plus integer Count of male patients aged 75 years and over
female_5_14 integer Count of female patients aged 5-14 years
female_15_24 integer Count of female patients aged 15-24 years
female_25_34 integer Count of female patients aged 25-34 years
female_35_44 integer Count of female patients aged 35-44 years
female_45_54 integer Count of female patients aged 45-54 years
female_55_64 integer Count of female patients aged 55-64 years
female_65_74 integer Count of female patients aged 65-74 years
female_75_plus integer Count of female patients aged 75 years and over
total_list_size integer Total count of patients registered at the practice
astro_pu_cost numeric ASTRO-PU (Age, Sex, and Temporary Resident Originated Prescribing Units) cost weighting
astro_pu_items numeric ASTRO-PU items weighting
star_pu character STAR-PU? as JSON?
pct_id character Primary Care Trust identifier (e.g., 08G)
practice character Practice code (e.g., E86005)
Rows 5 Columns 25
