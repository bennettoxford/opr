# opr: OpenPrescribing in R

Provides functions to connect to and query the OpenPrescribing.net
BigQuery database. Helps researchers understand the available data and
write readable, reusable, and modular queries for analysing English
prescribing data using 'dplyr' and 'dbplyr' instead of SQL.
